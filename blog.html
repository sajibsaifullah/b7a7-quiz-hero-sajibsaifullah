Some Important Q&A

Q.1: Write down the difference between 'map()', 'forEach()', 'filter()' & 'find()'.

'map()': It applies a provided function to every element of the array and returns a new array with the results of that function applied to each element in the original array. The original array remains unchanged.

'forEach()': It executes a provided function once for each element of the array. It does not return anything.

'filter()': It creates a new array with all elements that pass the test implemented by the provided function. The original array remains unchanged.

'find()': It returns the value of the first element in the array that satisfies the provided testing function. If no values satisfy the function, it returns undefined.

Q.2: Write down the difference between 'Local Storage', 'Session Storage' & 'Cookie'.

'Local Storage': LocalStorage is a data storage type of web storage. This allows the JavaScript sites and apps to store and access the data without any expiration date.

'Session Storage': Session storage allows you to store data in the browser depending on the system memory and the data stored in the browser until the browser is closed.

'Cookie': A cookie is an amount of information that persists between a server-side and a client-side.

Q.3: What is Event Loop?.

The event loop is like a traffic cop for JavaScript. It directs the flow of asynchronous operations by constantly checking if there are any pending tasks and processing them when the execution stack is empty, allowing non-blocking I/O operations to be performed without freezing the UI or blocking other code from executing.
